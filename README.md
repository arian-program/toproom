# toproom
it is private and secure chatroom in style of topsoft

## how to use it?
you can do it!

```bash
git clone https://github.com/arian-program/toproom
pip install -r txt.txt
```

### how to launch it?

you can do this !

```bash
./manage.py migrate
./manage.py runserver
```
